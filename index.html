How to Use

  1. Use the "Add Another Application" button to add new fields for your application details.
  2. Fill in the fields:
    • Application Name: Required. Enter the name of the application.
    • 32-bit URL: Optional. Provide the 32-bit version download link if available.
    • 64-bit URL: Optional. Provide the 64-bit version download link if available.
    • Arguments: Optional. Add installation arguments, such as silent install commands.
    • Tags: Required. Enter tags separated by commas (e.g., "developer, utility").
  3. Press "Generate JSON" to create the JSON output for your applications.
  4. Once the JSON is generated, use the "Download programs.json" button to download it.
